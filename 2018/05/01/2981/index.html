    Introduction

    備忘録。 /etc/fstab に下記のように書いてもマウントできなかった。 [code lang=”sh”] UUID=68ACE6B0ACE677C8 ~/media/hdd/windows ntfs default 0 0 [/code]

    Why?

    まぁお察しの通り、ホームディレクトリを表すチルダ(~)は、fstabがロードされる時点では、どのユーザに対応づけられているかわからないためでした。 同じ事を聞いている人がいました。
