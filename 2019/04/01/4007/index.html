    Introduction

    2019年最新のObject DetectionであるM2Detを使ってみます。
    目標は自作のデータセットを使うこと。
    まずはサンプルを実行してみます。

    How to

    公式ページはこちら。

    GitHub
    GitHub - VDIGPKU/M2Det: M2Det: A Single-Shot Object Detector based on Multi-Level Feature Pyramid Network
    https://github.com/qijiezhao/M2Det#Introduction

    なお

    • 推奨はAnacondaのようですが、ピュアなPythonで試します。
    • Python 3.6.7
    • CUDA 9.2

    という前提です。

    仮想環境構築

    仮想環境を作成しアクティベートします。

    環境変数設定

    CUDAの環境変数を設定します。
    設定済みならスキップ。

    設定後、下記のコマンドで確認を行います。

    パッケージのインストール

    M2DetはPyTorchの0.4.1をサポートと明言していますので従います。
    古いバージョンは下記のページから入手します。
    自分の環境のCUDAと合わせて適切なファイルをダウンロードします。

    PyTorch
    Previous PyTorch Versions
    https://pytorch.org/get-started/previous-versions/
    Installing previous versions of PyTorch

    今回はtorch-0.4.1-cp36-cp36m-linux_x86_64.whlになります。
    なお、途中の –cache-dir は、自環境でディスク容量不足のエラーが出たための対策です。

    インストール後、下記のようにしてPyTorchがCUDAを認識しているかどうかを確認します。

    ソースの入手

    pycocotoolsとnms

    ソースに付属のスクリプトを使ってpycocotoolsとnmsをインストールします。
