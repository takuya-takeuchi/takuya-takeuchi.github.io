    • 開発メモ その48 Red Hat Enterprise Linux 7.2にDockerをインストールする

    Problem

    .NET Coreを試したくなったけど、環境を汚すのが嫌なので、Dockerをインストールし、そこで遊んでみることにしました。 とは言っても、Dockerを触ったことはないので、インストール方法含めて、メモを残します。

    Solution

    インストール方法自体は、Redhatの公式ページに記載してあります。

    jp-redhat.com
    今日からできるDocker on RHEL7 | Docker on Red Hat Enterprise Linux 7
    http://jp-redhat.com/migration/docker/article.html
