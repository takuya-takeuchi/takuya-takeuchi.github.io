    Introduction

    Jetbrains Riderをインストールしました。

    ですが、しばらく使っていないうちに、正式版がリリースされたので、そのインストールをちょっとだけ。

    Preparation

    まずは、ダウンロード。
    2017/08の時点で2017.1が最新です。

    JetBrains
    Download Rider: The Cross-Platform .NET IDE
    https://www.jetbrains.com/rider/download/
    Download the latest version of Rider for Windows, macOS or Linux.

    ダウンロードしたら、端末を開きます。
    おそらく、ダウンロードフォルダにダウンロードしたファイルがあるので、ダウンロードフォルダに移動します。
    このあたりは前回と同じです。

    ダイアログが立ち上がり、設定を引き継ぐかどうかを尋ねられます。

    次にライセンス認証を要求されます。
    ライセンスを持っていないならトライアルが可能です。

    スプラッシュスクリーンです。

    メイン画面です。
