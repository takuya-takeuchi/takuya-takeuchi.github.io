    • 開発メモ その352 Remote Desktop で Ctrl + Alt + 矢印キーが動かない

    Introduction

    備忘録。
    マシンの前で直接、上記のキーを入力したらアプリ (WinForms) からの反応があった。

    ちなみにここでいう Remote Desktop は Windows に標準で入っている、いわゆる mstsc のこと。

    How to resolve?

    いつもの Stack Overflow…ではなく、Super User。
    まぁ、同じ Stack Exhange 配下だけど。

    要するに

    • Remote Desktop のバグ
      • Ctrl + Alt + 矢印キーが Remote Desktop によって占有されている
      • mstsc ではないが、Azure Virtual Desktop に接続できる Remote Desktop client for Windows では、この問題が修正されている
        • https://learn.microsoft.com/en-us/azure/virtual-desktop/whats-new-client-windows#updates-for-version-122924
    • ストア版の Microsoft Remote Desktop なら問題ない

    という感じ。

    ストア版で試したところ、確かに期待通りの動作をした。
