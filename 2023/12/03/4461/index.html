    Introduction

    備忘録。

    flutter で デバッグによるアタッチができず、2ヶ月くらい悩んでいた。
    経緯は

    1. iPhone の OS が古い (15) ので、16 の最新に更新
    2. Mac もアップデートしないといけなくなり、macOS Sonoma に更新
    3. アタッチできなくなる

    How to resolve?

    解決策は下記。

    Qiita
    実機ビルド時に警告が出て、アプリが起動するまで時間がかかった件【解決済み】 - Qiita

    だろうと思う。

    要するに

    1. rm -r ~/Library/Developer/Xcode/iOS\ DeviceSupport
    2. iPhone のデベロッパー機能をオフにして再度オンにする

    Github の Issue では、 1. の後に、 Xcode の Devices and Simulators からデバイスのペアを解除し、再度信頼すればいいみたいなこと書いてあった。

    自分は、上記の 2 ステップで解決した。
