    • Windows Server 2012 R2 #1 -ユーザー インターフェイスが戻せない-

    前回はWindows Server 2012 R2をデュアルブートでインストールしました。 MCP 70-410 Windows Server 2012 のインストールおよび構成を受験すべくMVA等で勉強を始めたのですが早速はまったので備忘録がてら残しておきます。

    Background

    Windows Server 2012 R2には下記の4つのユーザー インターフェイスが用意されています。

    なし

    グラフィカル管理ツールとインフラストラクチャ

    フル インストール オプション

    インストール済みデスクトップ エクスペリエンス機能

    それぞれの違いはWindows Server インストール オプションのリファレンス表を参照してください。 それで、自分はインストール時、フル インストール オプションでインストールを実行し、Server Core インストール オプション がどのようなものか分かりませんでした。 だから、インストール後変更できると知り、コマンドプロンプトから Powershell Uninstall-WindowsFeature Server-Gui-Shell -remove と叩いて、GUIが消えました。 で、一通り触った飽きた後、「さぁ戻すか」と思って、 Powershell Install-WindowsFeature Server-Gui-Shell とやったが戻らない＼(^o^)／ 黒背景のコマンドプロンプトに赤文字でエラーの旨が表示され恐怖心MAX。 片っ端から調べてみたところ、どうもインストールメディアが必要とのこと。 インストールメディアが無くても出来てる人がネットに居るのは何故？

    Resolution

    結論から言えば、インストールメディアを挿入した状態で Powershell Install-WindowsFeature Server-Gui-Shell –Restart –Source wim:D:\source\install.wim:4 と入力すれば戻ります。 ※D:\source\の部分はインストールメディアのドライブに応じて変更 初心者にServerOSは厳しい道のりなのか…
