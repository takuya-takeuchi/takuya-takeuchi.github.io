    • 開発メモ その389 Visual Studio Code が起動しない

    Introduction

    掲題の通り。

    タスクバーあるいはデスクトップのアイコンから起動するも即座に終了してしまう。
    イベントビューアを見ても何もエラーが出ていないので、クラッシュしているのではなく、何かしらエラーが発生して正常に終了している模様。

    How to resolve?

    結論から言えば再インストールして解決した。
    別にアプリをアンインストールすることもなく、公式ページからインストーラを落としてきてインストールしただけ。
    少しインストール済みのバージョンが古かったのかアップデート扱いでインストールされた。

    ちなみにコマンドプロンプトから起動すると、エラーの内容が出力されていた。

    どういうわけか、 JavaScript エンジン　V8 に問題が起きていた。
    公式でも同様の問題 Cant open installed vs code program from windows #178373 が上がっていて、再インストールを提案されて、それで事象は解決していた。
