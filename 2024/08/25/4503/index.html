    • 開発メモ その395 PXE ブートで Ubuntu 22.04 が起動できない (unable to find a live file system on the network)

    Introduction

    PXE ブートができるようになり、Ubuntu 22.04 を試したところ遭遇したのが掲題のエラー。

    How to resolve?

    最新の 22.04.4 の iso なら問題がなかった。
    また、ubuntu jammy 22.04 iso: unable to find a live file system on the network として報告もある。

    コメントを見ていると、古い initrd や iso を使っていたという記述もあるが、

    Hi, I just ran into exactly this problem with the 22.04.3 ISO freshly downloaded today.

    とあったので、22.04.4 を使ったら問題が解決した。
