    Introduction

    会社で使っているマシンには、有線 LAN が二つ付いてる。

    マザーボードに備わっているものと USB ケーブルで追加したもの。

    後者はよく IP を変更したり柔軟に使って重宝しているのだが、つい最近、この IP を元に戻せなくなった事があった。

    IP のプロパティで元の IP を設定しても、下記の状態に戻ってしまうのである。

    ちなみに、別の IP なら問題ないのだが、会社で使う IP なのでおいそれと自由な値は使えない。
