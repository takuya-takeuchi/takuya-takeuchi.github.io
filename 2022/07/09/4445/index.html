    How to resolve?

    上記にあるように、.NET 5 でプロジェクトを作成し、手動で .NET 6 に更新するという手段しかない模様。

    作成された .csproj を開き、TargetFramework を変更。

    必要に応じて Implicit using directives (暗黙的な using ディレクティブ) や Nullable (Null許容) を有効にすれば完成。
