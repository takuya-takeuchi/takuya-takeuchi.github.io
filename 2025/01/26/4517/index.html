    Introduction

    久しぶりに ios アプリの検証をしていると…

    なぜ？

    What’s wrong?

    こういう時は Xcode でプロジェクトを開き、ビルドを試してみるのが有効。
    というこで開くと…

    PLA (Program License Agreement) に同意していないから、Apple Developer Program のリソースにアクセスできないとのこと。
    Web サイトで Apple Developer Program にログインすると、PLA への同意を求められていることがわかります。

    Review agreement を押下し、同意します。
    同意が Xcode から Try Again しないと fltter build は通らないままでしたので注意。
