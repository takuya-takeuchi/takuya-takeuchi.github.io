    • 開発メモ その456 iOS/Android アプリでクレジットカード番号を自動入力できない

    Introduction

    備忘録。
    できない、という情報を集めた。

    Evidences

    • 40609671: UITextContentType creditCardNumber non-functional #19779

    2018年5月29日の Issue。
    環境は

    • iOS: 11.3
    • Xcode: 9.3
    • macOS: High Sierra 10.13.4

    iOS シミュレータと物理デバイスでも、保存されているクレジットカードの自動入力提案が動作しない。

    • AutoFillHints doesn’t pull up Credit Card info #98762

    2022年2月19日の Issue。
    環境は

    • flutter: 2.10.1
    • Xcode: 13.2.1
    • Android: 不明
    • Android Studio: 2020.3
    • iOS: 14.4.1, 15.2

    コメントで、クレジットカード情報は Safari 専用、とあるので iOS アプリでは動かない可能性が高い。

    • Android Autofill not working on latest stable (2.10.3) #100618

    2022年5月23日の Issue。
    環境は

    • flutter: 2.10.3
    • Android: 不明

    クレジットカード番号以外の一部の自動入力が動作していない。

    • iOS TextField with TextInputType.number and AutofillHints.creditCardNumber shows the wrong keyboard #104604

    2022年5月25日の Issue。
    環境は

    • flutter: 3.0.1, 3.22.0
    • Xcode: 13.3.1
    • Android Studio: 2021.2

    TextInputType.number と AutofillHints.creditCardNumber を使用すると表示されるキーボードがおかしい。
    途中のコメントでクレジットカードの自動入力は現在サポートされていない、とある (コメントの根拠となるドキュメントなどの提示は無し)

    • How to autofill credit/debit-card details in inputfield in WebView in RN iOS app? #2990

    2023年6月7日の Issue。
    環境は

    • React Native: 不明

    flutter ではないが、同様の事象が React Native でも発生している。
    ポイントは Safari ブラウザや Android の Google Pay は動作しているが、iOS アプリの WebView では動作しない、という点。

    • (iOS) autoComplete=”cc-number” is not doing anything to allow autofill #36089

    2023年2月8日の Issue。
    環境は

    • React Native: 0.69.0, 0.72.5, 0.73.4, 0.74, 0.76.1
    • macOS: 15.1

    こちらも flutter ではなく、React Native であるが、Android の Web アプリでは動作するが、iOS は動かない、という同様の問題。
    ここで、有用なコメントがあり

    • SwiftUI の最小コードでも動作しない
    • Apple WWDC のデモでもクレジットカードの自動入力だけは言及していない
      • AutoFill everywhere
    • 公式ページにおいて、支払内情報の自動入力に関して、Safari 以外には一切言及していない
      • iPhoneのSafariで情報を自動入力する の 支払い情報を自動的に入力する を参照
    • 世の中の多くのアプリがクレジットカード番号の自動入力を実装していない
      • Paypal
      • Cash App
      • Venmo
      • Coinbase

    特に最後のコメントが説得力がある。

    実際に、上記の最小コードを動かしてみたところ、やはり動かなかった。
