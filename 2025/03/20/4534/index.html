    • 開発メモ その427 iOS アプリを Xcode でビルドするとキーチェーンへのアクセスが何度も行われる

    Introduction

    備忘録。
    自分の環境では遭遇したことは無いのだが、客先で作業しているときにこの事象が発生した。
    下記のダイアログが繰り返し表示される。

    )

    常に許可を指定しても、ダイアログがひたすら繰り返して表示される。

    How to resolve?

    この事象は沢山報告例がある。

    • Codesign wants to access key “access” in your keychain, I put in my login password but keeps asking me
    • [macOS] キーチェーンのパスワードを急に要求しだした

    解決策は、iOS アプリの署名に使用する秘密鍵に対して codesign をアクセス制御の許可するアプリ一覧に追加する、というものだ。

    これ自体はシンプルな解決策だが、問題は codesign を追加する方法が書かれていない点にある。

    )

    [+] を押下して、ダイアログを開く。

    )

    shift + command (VNC 経由の Windows の場合 Alt) + G で「フォルダへ移動」ダイアログを開く。

    )

    /usr/bin と入力して Enter を押下。

    )

    /usr/bin のフォルダが選択されるので目的の codesign を選択して追加できる。

    Mac をメインで使っているユーザなら当たり前なのかもしれないが、Windows をメインにしているユーザだと、こういったちょっとしたノウハウが致命的になる。
